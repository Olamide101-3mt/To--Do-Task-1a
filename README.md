# Advanced Todo Card

A simple, interactive todo card component built with HTML, CSS, and JavaScript. This project demonstrates advanced UI interactions including collapsible content, real-time status updates, priority indicators, and time-based calculations.

## Features

- **Priority Indicators**: Visual border colors based on task priority (High, Medium, Low)
- **Collapsible Description**: Expand/collapse long task descriptions
- **Status Management**: Toggle between Pending, In Progress, and Done states
- **Real-time Time Calculations**: Shows time remaining or overdue status
- **Edit Mode**: Inline editing of task details
- **Responsive Design**: Works on various screen sizes

## How to Run

1. Clone or download the project files
2. Open `index.html` in any modern web browser
3. The todo card will load with sample data

No server or build process is required - it's a static web page.

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - JavaScript functionality

## Browser Support

Works in all modern browsers that support ES6+ JavaScript features.

## Customization

You can modify the initial task data in `script.js` to change the default todo item, or extend the functionality as needed.
